# discounted-cash-flow-valuation-project

This project presents a discounted cash flow valuation model built in Excel. The model estimates enterprise value and implied equity value using projected free cash flows, weighted average cost of capital, terminal value assumptions, and sensitivity analysis.

## Tools Used
- Microsoft Excel
- PowerPoint
- Financial modeling
- DCF valuation
- Sensitivity analysis

## Key Components
- Revenue and margin projections
- Free cash flow forecast
- WACC calculation
- Terminal value calculation
- Enterprise value and equity value bridge
- Sensitivity analysis
- Investment recommendation presentation

## Files
- `DCF_Model.xlsx`: Full Excel valuation model
- `DCF_Presentation.pptx`: Summary presentation of valuation assumptions, outputs, and recommendation
